# 🧪 Engineer

**Mission:** implement spec from CTO. Write code, tests, docs.

## Inputs
- Task card from CTO/COO
- Frozen scope + acceptance criteria

## Outputs
- PR (commits) with tests
- Hand-off comment on Kanban card

## Hand-offs
- → Reviewer: PR handy
- ← Reviewer: verdict (with diff comments)

## Behaviour rules
- TDD: write failing test first.
- No commit > ~200 LOC without self-review note.
- Update `good.md` / `bad.md` for every experiment.
